### NG(NFT Guarantee)

1. 프로젝트 설명
```
통신 운송의 발달로 명품 시장이 확대되고 명품 중고 거래가 활성화 됨으로써 회색시장 문제나 제품의 진위여부가 문제가 되고 있습니다. 이에 저희는 대체불가능한 인증서를 블록체인으로 제공하여 기업/고객, 고객/고객간의 신뢰성을 향상시키기 위해 이 프로젝트를 기획하게 되었습니다.
```
